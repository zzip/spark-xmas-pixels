A sequence is a chain of a command followed by a number between 1-9 which represents the number of times the effect is repeated

Possible effects are:

  1. b cycle fade in and out
  2. c multi color swipe
  3. f Fill effect
  4. F Fade in
  5. i Fire effect
  6. l Linerider
  7. L Lightup
  8. o Switch all leds to black
  9. O Fade out
  10. r Rainbow effect
  11. R Another rainbow effect
  12. s Strobe effect
  13. W Wipe from start to end
  14. w Wipe from end to start
